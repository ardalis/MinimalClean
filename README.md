# MinimalClean

A minimal ASP.NET Core Aspire Clean Architecture with Vertical Slice Architecture solution template.

This implementation has been moved to a subfolder in my Clean Architecture template, [here](https://github.com/ardalis/CleanArchitecture/tree/main/MinimalClean).

You can install and run it via `dotnet` like this:

```bash
# Minimal Clean Architecture template  
dotnet new install Ardalis.MinimalClean.Template

dotnet new min-clean -o Your.ProjectName
```

I've found this single-project approach works well for modules within a [modular monolith architecture](https://modularmonoliths.com/).
